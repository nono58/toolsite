FAQ

What is URL encoding?
URL encoding (percent-encoding) replaces characters that aren't allowed in URLs with a "%" followed by two hex digits. For example, a space becomes %20 and & becomes %26.
When do I need to URL-encode?
When passing special characters in query parameters, form data, or API requests. Characters like spaces, &, =, #, and non-ASCII text must be encoded.
What's the difference between encodeURI and encodeURIComponent?
This tool uses encodeURIComponent, which encodes everything except A-Z a-z 0-9 - _ . ! ~ * ' ( ). It's suitable for encoding individual query parameter values.
