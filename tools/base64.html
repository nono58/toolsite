FAQ

What is Base64?
Base64 is an encoding scheme that converts binary data into ASCII text. It's commonly used to embed images in HTML/CSS, transmit data in JSON, or store binary data in text-based formats.
Is Base64 encryption?
No. Base64 is encoding, not encryption. Anyone can decode a Base64 string without a key. Do not use it to secure sensitive data.
Does this tool support Unicode?
Yes. Text is encoded as UTF-8 before Base64 encoding, so emojis and non-ASCII characters are fully supported.
